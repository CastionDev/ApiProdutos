# API de Produtos

API REST desenvolvida com ASP.NET Core para gerenciamento de produtos.

## Funcionalidades

- CRUD completo de produtos
- Validação de dados com DTO
- Integração com SQLite (Entity Framework Core)
- Documentação com Swagger

## Tecnologias

- .NET 8
- ASP.NET Core
- Entity Framework Core
- SQLite

## Como executar

1. Clonar o repositório
2. Executar o projeto
3. Acessar o Swagger em:
   https://localhost:7128/swagger
